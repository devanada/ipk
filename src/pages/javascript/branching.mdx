# JavaScript | Branching

## Overview

JavaScript adalah bahasa pemrograman yang sangat populer di dunia web. Dengan kemampuannya yang kuat dan fleksibel, JavaScript memungkinkan pengembang untuk membuat aplikasi yang interaktif dan dinamis. Salah satu konsep penting dalam JavaScript adalah percabangan, yang memungkinkan program untuk mengambil keputusan berdasarkan kondisi tertentu.

Percabangan dalam JavaScript merupakan suatu cara untuk menjalankan blok kode tertentu berdasarkan kondisi yang ditentukan. Hal ini memungkinkan kita untuk membuat aplikasi dengan logika yang kompleks dan responsif terhadap input pengguna. Memahami konsep percabangan akan membantu kita mengembangkan kode JavaScript dengan lebih efisien dan terstruktur.

Ketika kita mempelajari percabangan dalam JavaScript, ada beberapa hal yang perlu dipahami. Pertama, kita perlu mengetahui statement if-else, yaitu salah satu cara dasar untuk melakukan percabangan dalam JavaScript. Statement if-else digunakan ketika ingin menjalankan bagian kode tertentu jika suatu kondisi bernilai true atau false.

Namun, selain statement if-else, terdapat juga alternatif lain seperti switch statement. Switch statement berguna ketika kita memiliki banyak kondisi yang mungkin terjadi dan ingin menjalankan blok kode spesifik berdasarkan nilai dari sebuah ekspresi.

## Mengenal Statement If dalam JavaScript

Statement If merupakan salah satu konsep dasar dalam percabangan JavaScript yang digunakan untuk mengarahkan aliran program berdasarkan kondisi tertentu. Dalam konteks ini, kondisi adalah sebuah ekspresi yang mengembalikan nilai true atau false. Jika kondisi bernilai true, maka blok kode yang terdapat di dalam statement If akan dieksekusi, sedangkan jika kondisi bernilai false, blok kode tersebut akan dilewatkan.

Penerapan statement If memungkinkan kita untuk membuat keputusan program berdasarkan kondisi yang ada. Sebagai contoh, jika kita ingin menampilkan pesan "Selamat pagi!" hanya pada pukul 6-11 pagi, maka kita dapat menggunakan statement If dengan memeriksa waktu saat ini dan menentukan apakah waktu tersebut berada dalam rentang yang diinginkan.

Sintaks dasar dari statement If adalah sebagai berikut:

```javascript
if (kondisi) {
  //blok kode yang akan dieksekusi jika kondisi bernilai true
}
```

Penting untuk mencatat bahwa blok kode di dalam if harus diapit oleh kurung kurawal ("{}") untuk menjaga kelangsungan operasi dan membantu kejelasan struktur kode. Jika hanya ada satu pernyataan di dalam blok if, kurung kurawal bisa tidak ditulis, tetapi disarankan agar tetap digunakan guna menjaga konsistensi dan kemudahan pemeliharaan kode.

## Memahami Konsep If Else dalam Percabangan JavaScript

Dalam bahasa pemrograman JavaScript, salah satu cara yang paling umum untuk melakukan percabangan adalah menggunakan konsep If Else. Konsep ini sangat penting untuk dipahami karena memberikan kemampuan kepada kita untuk membuat keputusan berdasarkan kondisi tertentu.

```javascript
if (kondisi) {
  //blok kode yang akan dieksekusi jika kondisi bernilai true
} else {
  //blok kode yang akan dieksekusi jika kondisi bernilai false
}
```

Statement If Else digunakan ketika kita ingin menjalankan blok kode tertentu jika suatu kondisi tertentu terpenuhi. Namun, jika kondisi tersebut tidak terpenuhi, kita dapat menjalankan blok kode alternatif menggunakan statement else.

If Else dalam JavaScript memungkinkan kita untuk membuat program yang responsif dan adaptif. Kita dapat mengatur alur logika program dengan memberikan instruksi yang berbeda tergantung pada nilai dari suatu variabel atau hasil evaluasi ekspresi.

Penerapan konsep If Else dapat menghasilkan kode yang jauh lebih fleksibel dan efisien. Dengan memahami penggunaannya dengan baik, kita dapat menghindari melakukan tindakan yang tidak perlu dan mempercepat eksekusi program.

## Percabangan Switch: Alternatif untuk If Else

Salah satu alternatif yang bisa digunakan dalam percabangan JavaScript adalah menggunakan statement switch. Switch digunakan ketika terdapat banyak kondisi yang harus dievaluasi dan tindakan yang berbeda-beda akan diambil berdasarkan nilai dari suatu ekspresi.

Dibandingkan dengan if else, switch sering kali memberikan kesan kode yang lebih terstruktur dan mudah dibaca. Ia memisahkan blok kode untuk setiap kasus tertentu, memungkinkan pemrogram untuk dengan jelas melihat tindakan apa yang akan diambil berdasarkan nilai ekspresi.

Menggunakan switch dalam percabangan juga dapat meningkatkan efisiensi dalam penggunaan memori dan waktu eksekusi kode. Ketika ada banyak kondisi, if else harus mengevaluasi setiap kondisi secara berurutan, sedangkan switch langsung melompat ke kasus yang sesuai, menghindari pengecekan kondisi berlebihan.

Perhatikan contoh penggunaan switch di bawah ini:

```javascript
let hari = 2;
let namaHari;

switch (hari) {
  case 1:
    namaHari = "Senin";
    break;
  case 2:
    namaHari = "Selasa";
    break;
  case 3:
    namaHari = "Rabu";
    break;
  default:
    namaHari = "Hari tidak valid";
}

console.log(namaHari); // Output: Selasa
```

Dalam contoh di atas, kita menggunakan variabel 'hari' sebagai ekspresi untuk mencari tahu nama hari yang sesuai. Switch akan mengevaluasi ekspresi tersebut dan melompat ke kasus yang cocok. Jika nilai 'hari' adalah 2, maka 'namaHari' akan diisi dengan string "Selasa". Jika nilai 'hari' tidak cocok dengan kasus manapun, maka 'namaHari' akan diisi dengan string "Hari tidak valid" melalui blok default.

## Menjelajahi Percabangan Bersarang dalam JavaScript

Saat bekerja dengan logika yang kompleks, sering kali kita akan menghadapi situasi di mana kita perlu menggunakan percabangan bersarang dalam JavaScript. Percabangan bersarang adalah ketika kita menggunakan statement if atau switch di dalam blok statement if atau switch lainnya. Melalui teknik ini, kita dapat menciptakan logika yang lebih rumit dan fleksibel.

Salah satu contoh penggunaan percabangan bersarang adalah dalam validasi data input pada sebuah form. Misalkan kita memiliki form dengan beberapa field yang harus diisi oleh pengguna. Kita dapat menggunakan percabangan bersarang untuk memeriksa satu per satu field tersebut, dan memberikan pesan kesalahan jika ada field yang tidak diisi.

Saat menjelajahi percabangan bersarang, penting untuk mempertimbangkan urutan evaluasi kondisi-kondisi tersebut. JavaScript akan mengevaluasi kondisi secara berurutan dari atas ke bawah, dan hanya akan menjalankan blok kode dari kondisi pertama yang bernilai true. Oleh karena itu, urutan penulisan kondisi-kondisi sangat krusial untuk mencapai hasil yang diinginkan.

Di samping itu, menggunakan terlalu banyak level percabangan bersarang dapat membuat kode menjadi sulit dibaca dan dipahami. Oleh karena itu, sebaiknya gunakan teknik ini dengan bijak dan hanya ketika diperlukan. Jika mungkin, pertimbangkanlah alternatif lain seperti function atau modularisasi kode untuk menghindari kompleksitas yang berlebihan.

## Menggunakan Statement If dalam Percabangan Bersarang

Dalam pemrograman JavaScript, percabangan bersarang adalah teknik yang sangat bermanfaat ketika kita perlu melakukan beberapa pengecekan kondisi secara berurutan. Salah satu cara untuk menerapkan percabangan bersarang adalah dengan menggunakan statement if dalam struktur percabangan.

Statement if dalam percabangan bersarang memungkinkan kita untuk mengevaluasi beberapa kondisi dan mengambil tindakan yang sesuai tergantung pada hasilnya. Kita dapat meletakkan statement if di dalam blok if lainnya, sehingga menciptakan struktur percabangan yang kompleks dan fleksibel.

Contoh penggunaan statement if dalam percabangan bersarang dapat ditemukan dalam skenario seperti validasi input pengguna. Misalkan kita ingin memeriksa apakah pengguna telah memasukkan nilai angka positif yang lebih besar dari 10, sebelum menjalankan fungsi tertentu. Pertama, kita bisa menggunakan sebuah statement if untuk memeriksa apakah angka tersebut positif. Jika benar, kita kemudian bisa menggunakan statement if kedua untuk memeriksa apakah angka tersebut juga lebih besar dari 10.

Penggunaan statement if dalam percabangan bersarang memberikan fleksibilitas dan kontrol tambahan terhadap alur program. Namun, kita perlu berhati-hati agar tidak membuat struktur kode yang terlalu kompleks atau rumit. Dengan merencanakan dengan baik dan menyusun logika dengan jelas, kita dapat menciptakan kode yang mudah dibaca dan dipelihara.

## Mengkombinasikan If Else dalam Percabangan Bersarang

Ketika kita berhadapan dengan situasi yang lebih kompleks, sering kali kita perlu menggabungkan kondisi if else dalam percabangan JavaScript. Dengan mengkombinasikan if else, kita dapat membuat logika yang lebih lanjut untuk menangani berbagai kemungkinan hasil.

Salah satu contoh penggunaan kombinasi if else adalah ketika kita ingin memeriksa beberapa kondisi sekaligus dan menentukan tindakan yang sesuai berdasarkan kondisi tersebut. Misalnya, dalam sebuah aplikasi manajemen toko online, kita ingin memberikan diskon kepada pelanggan berdasarkan total belanja mereka.

Dalam kasus ini, kita dapat menggunakan percabangan bersarang untuk memeriksa total belanja pelanggan. Jika total belanja melebihi batas tertentu, pelanggan akan mendapatkan diskon 10%. Namun, jika total belanja tidak mencapai batas tersebut tetapi melebihi batas minimum untuk mendapatkan diskon 5%, maka pelanggan akan mendapatkan diskon 5%. Jika tidak memenuhi kedua kondisi tersebut, pelanggan tidak akan mendapatkan diskon sama sekali.

Kita dapat mencapai ini dengan menggunakn kombinasi if else dalam percabangan bersarang. Pertama-tama, kita periksa apakah total belanja lebih besar dari batas tertentu menggunakan statement if. Jika iya, maka tindakan yang diambil adalah memberikan diskon 10%. Namun jika tidak memenuhi kondisi pertama dan memenuhi kondisi kedua, kita menggunakan else if untuk memeriksa apakah total belanja melebihi batas minimum untuk diskon 5%. Jika iya, maka tindakan yang diambil adalah memberikan diskon 5%. Jika tidak memenuhi kedua kondisi tersebut, maka tindakan default adalah tidak memberikan diskon sama sekali.
