# Installation

## Overview

Memiliki perangkat PC ataupun laptop tidaklah cukup untuk memulai perjalanan kita menjadi seorang Frontend Engineer, untuk itu dibutuhkan beberapa tools pembantu yang perlu di-install terlebih dahulu.

Pada dokumentasi ini akan dijelaskan beberapa tools yang perlu kalian pasang di perangkat kalian dan juga step by step untuk membantu kamu meng-install tools ini di perangkat kalian.

## Git

Git adalah sistem kontrol versi yang sangat penting untuk mengelola kode sumber. Kamu dapat mengunduh dan menginstal Git dengan langkah-langkah berikut:

  <Tab>
    <Steps>
        ### Unduh Git
        
        Kunjungi situs resmi Git di https://git-scm.com/downloads dan
        pilih versi Git yang sesuai dengan sistem operasimu (Windows, macOS, atau
        Linux).

        Jalankan installer yang telah Kamu unduh, ikuti

        memeriksa kotak `Use Git from the Windows Command Prompt` jika Kamu

        Setelah instalasi selesai,

        memastikan Git telah diinstal dengan benar:

        git --version

## Visual Studio Code (VSCode)

Visual Studio Code adalah salah satu editor kode sumber terbaik yang digunakan oleh banyak pengembang. Berikut adalah langkah-langkah untuk menginstalnya:

1. Unduh VSCode: Kunjungi situs resmi Visual Studio Code di https://code.visualstudio.com/ dan unduh versi yang sesuai dengan sistem operasimu (Windows, macOS, atau Linux).

2. Instalasi VSCode: Jalankan installer yang telah Kamu unduh dan ikuti petunjuknya. VSCode akan diinstal dengan konfigurasi default.

3. Ekstensi VSCode: VSCode memiliki berbagai ekstensi yang berguna untuk pengembangan web. Kamu dapat menginstal ekstensi dengan membuka VSCode, pergi ke Marketplace (Ctrl+Shift+X) dan mencari ekstensi yang Kamu butuhkan.

## Node.js

Node.js adalah lingkungan JavaScript yang memungkinkan kita untuk menjalankan kode JavaScript di sisi server. Berikut adalah langkah-langkahnya:

1. Unduh Node.js: Kunjungi situs resmi Node.js di https://nodejs.org/ dan unduh versi LTS (Long Term Support) yang sesuai dengan sistem operasimu. LTS adalah versi yang stabil dan direkomendasikan.

2. Instalasi Node.js: Jalankan installer yang telah Kamu unduh, ikuti petunjuknya, dan pilih opsi yang sesuai. Node.js dan npm akan diinstal bersamaan.

3. Verifikasi Instalasi: Setelah instalasi selesai, buka terminal atau command prompt dan ketik perintah berikut untuk memeriksa versi Node.js dan npm:

```bash
node -v
npm -v
```

Kamu akan melihat versi Node.js dan npm yang terpasang.

Setelah mengikuti langkah-langkah di atas, Kamu akan memiliki lingkungan pengembangan yang lengkap untuk menjadi seorang Frontend Engineer. Selamat belajar dan mengembangkan aplikasi web! Jangan lupa untuk memperdalam pengetahuan Kamu tentang HTML, CSS, dan JavaScript serta memahami konsep frontend development yang lebih dalam.
